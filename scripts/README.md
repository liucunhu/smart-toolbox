# Smart-Toolbox 脚本集合

本文件夹包含项目的所有测试脚本、调试脚本和工具脚本。

## 📁 脚本分类

### 测试脚本 (`test_*.py`)
- 功能测试脚本
- 集成测试脚本
- 自动化测试脚本

### 创建脚本 (`create_*.py`)
- 账号创建脚本
- 功能创建脚本
- 数据初始化脚本

### 调试脚本 (`debug_*.py`, `diagnose_*.py`)
- 问题诊断脚本
- 日志分析脚本
- 错误排查脚本

### 修复脚本 (`fix_*.py`, `alter_*.py`)
- 数据修复脚本
- 配置修复脚本
- 枚举修复脚本

### 验证脚本 (`verify_*.py`, `check_*.py`)
- 功能验证脚本
- 状态检查脚本
- 连接测试脚本

### 分析脚本 (`analyze_*.py`, `find_*.py`, `inspect_*.py`)
- 结构分析脚本
- 元素查找脚本
- 页面检查脚本

### PowerShell脚本 (`*.ps1`)
- 服务启动脚本
- 测试运行脚本
- 批量处理脚本

### SQL脚本 (`*.sql`)
- 数据库修复脚本
- 数据迁移脚本

---

## ⚠️ 使用说明

1. **测试脚本**: 运行前确保后端服务已启动
2. **创建脚本**: 会修改数据库，请谨慎使用
3. **调试脚本**: 需要Playwright已安装
4. **修复脚本**: 执行前建议备份数据库

---

**说明**: 所有脚本统一归档至此文件夹，方便管理和查找。
