# 网络素材搜索与二次原创功能

## 📋 功能说明

该功能可以在生成文章前自动从互联网获取相关主题的实时素材，然后基于这些素材进行**二次原创**，大大提升文章的时效性和质量。

## 🔧 配置步骤

### 1. 选择搜索引擎（三选一）

#### 方案A: Bing Search API（推荐）
- **优点**: 稳定可靠，中文支持好
- **缺点**: 需要付费订阅
- **价格**: $25/月（每月3000次查询）

**获取API Key:**
1. 访问 https://portal.azure.com
2. 创建 "Bing Search v7" 资源
3. 复制 API Key

**配置 `.env`:**
```bash
BING_SEARCH_API_KEY=your_bing_api_key_here
```

---

#### 方案B: SerpAPI
- **优点**: 支持Google/Bing/Baidu等多个引擎
- **缺点**: 付费服务
- **价格**: $50/月（每月5000次查询）

**获取API Key:**
1. 访问 https://serpapi.com
2. 注册账号并获取API Key

**配置 `.env`:**
```bash
SERPAPI_API_KEY=your_serpapi_key_here
```

---

#### 方案C: 不使用网络搜索（默认）
如果不配置任何搜索引擎，系统将仅使用LLM的内置知识生成内容。

## 🚀 使用方法

### API调用示例

```python
from app.services.content.copywriting_generation import CopywritingGenerator

# 创建生成器
generator = CopywritingGenerator()

# 生成文章（默认启用网络搜索）
result = generator.generate_script(
    platform="toutiao",
    topic="人工智能技术在2026年的最新突破",
    enable_web_search=True  # 启用网络搜索
)

if result:
    print(f"标题: {result['title']}")
    print(f"内容: {result['content'][:200]}...")
```

### 禁用网络搜索

如果只想使用LLM的内置知识，可以禁用网络搜索：

```python
result = generator.generate_script(
    platform="toutiao",
    topic="人工智能技术",
    enable_web_search=False  # 禁用网络搜索
)
```

## 📊 工作流程

```
1. 用户输入主题: "人工智能技术在2026年的最新突破"
         ↓
2. 网络搜索获取素材 (5条结果)
   - 搜索结果1: 标题 + 摘要
   - 搜索结果2: 标题 + 摘要
   - ...
         ↓
3. 格式化素材为Prompt
   【网络素材参考】
   1. xxx标题
      xxx摘要
   
   2. xxx标题
      xxx摘要
         ↓
4. LLM基于素材二次原创
   - 深度加工和重新组织
   - 结合最新信息和数据
   - 加入观点和分枃
   - 保持原创性
         ↓
5. 输出高质量文章
```

## 💡 最佳实践

### 1. 选择合适的主题
✅ **好的主题**（有时效性）:
- "2026年人工智能最新突破"
- "DeepSeek V4发布后的行业影响"
- "春节后旅游市场复苏情况"

❌ **不好的主题**（过于宽泛）:
- "人工智能"
- "科技"
- "生活"

### 2. 验证搜索结果
系统会显示获取到的素材数量和质量：
```
🌐 开始搜索网络素材: 人工智能技术在2026年的最新突破
✅ 获取到 5 条网络素材
```

如果显示：
```
⚠️  未获取到网络素材，将仅使用LLM知识
```
建议检查：
- API Key是否正确配置
- 网络连接是否正常
- 主题是否过于冷门

### 3. 调整搜索参数

在 `web_search.py` 中可以调整：
```python
# 修改搜索结果数量
results = await search_service.search_materials(
    topic, 
    num_results=10  # 增加结果数量
)

# 修改超时时间
self.timeout = 15.0  # 增加到15秒
```

## ⚠️ 注意事项

### 1. 原创性保证
- ✅ 系统会提示LLM不要直接复制素材
- ✅ 要求进行深度加工和重新组织
- ✅ 加入自己的观点和分析
- ⚠️ 但仍需人工审核，确保无抄袭风险

### 2. 性能影响
- 网络搜索会增加 **5-15秒** 的耗时
- 如果不需要实时信息，可以禁用此功能

### 3. API费用
- Bing Search: $0.008/次查询
- SerpAPI: $0.01/次查询
- 建议设置预算上限

## 🔍 故障排查

### 问题1: 搜索失败
```
⚠️  网络搜索失败: Connection timeout
```

**解决:**
1. 检查网络连接
2. 验证API Key是否正确
3. 增加超时时间

### 问题2: 未获取到素材
```
⚠️  未获取到网络素材，将仅使用LLM知识
```

**解决:**
1. 尝试更换主题关键词
2. 检查搜索引擎配额是否用完
3. 切换到其他搜索引擎

### 问题3: 文章内容与素材无关
**原因:** LLM可能没有充分利用素材

**解决:**
1. 增强Prompt中的创作要求
2. 提供更多相关素材（增加num_results）
3. 调整temperature参数（降低到0.6）
