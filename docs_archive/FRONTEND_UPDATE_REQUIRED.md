## 🔧 如何查看更新

### 方法1: 强制刷新浏览器（推荐）

1. 打开浏览器访问: `http://localhost:3002/toutiao/account`
2. 按以下快捷键强制刷新:
   - **Windows/Linux**: `Ctrl + Shift + R` 或 `Ctrl + F5`
   - **Mac**: `Cmd + Shift + R`

### 方法2: 清除缓存后刷新

1. 打开浏览器开发者工具 (`F12`)
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

### 方法3: 重启前端服务

如果上述方法无效，可以重启前端服务：

```powershell
# 停止当前服务（Ctrl+C）
# 然后重新启动
cd frontend
npm run dev
```

## 📊 技术细节

### 后端API变更

**旧接口调用**:
```javascript
POST /api/v1/accounts/toutiao/login?account_id=1&username=xxx&password=xxx
```

**新接口调用**:
```javascript
// 方式1: 只提供用户名密码（推荐）
POST /api/v1/accounts/toutiao/login?username=xxx&password=xxx

// 方式2: 提供账号ID（向后兼容）
POST /api/v1/accounts/toutiao/login?account_id=1
```

### 前端代码变更

**移除的代码**:
```typescript
// ❌ 删除
const form = ref({
  accountId: 1,  // 不再需要
  username: '',
  password: ''
})
```

**新增的代码**:
```typescript
// ✅ 添加
const currentAccountId = ref<number | null>(null)

// 登录后保存账号ID
if (response.data.status === 'success') {
  currentAccountId.value = response.data.account_id || null
}
```

## 🐛 如果还是看到旧界面

### 问题1: Vite热重载未生效

**解决方案**:
```bash
# 1. 停止前端服务（Ctrl+C）
# 2. 清除Vite缓存
cd frontend
rm -rf node_modules/.vite

# 3. 重新启动
npm run dev
```

### 问题2: 浏览器缓存

**解决方案**:
1. 打开浏览器设置
2. 清除浏览数据
3. 选择"缓存的图片和文件"
4. 点击"清除数据"
5. 重新访问页面

### 问题3: Service Worker缓存

**解决方案**:
1. 打开开发者工具 (`F12`)
2. 切换到 Application 标签
3. 找到 Service Workers
4. 点击 "Unregister"
5. 刷新页面
