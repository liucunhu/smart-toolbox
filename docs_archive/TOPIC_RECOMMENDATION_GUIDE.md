# 智能主题推荐功能使用指南

## 📋 功能概述

一键发布时，系统会**自动推荐当前最有可能火的主题**，结合：
- 🔥 **实时热搜数据**：各平台最新热点
- 📊 **历史数据分析**：账号过往表现优秀的主题类型
- 🎯 **个性化算法**：为每个账号定制推荐

## 🎯 推荐算法原理

### 1. 数据来源

#### A. 实时热搜（权重60%）
- 抖音热搜榜
- 今日头条热榜
- B站热门视频
- 小红书发现页

#### B. 历史表现（权重40%）
- 该账号过去20篇文章的数据
- 阅读量最高的主题类型
- 互动率最好的内容方向

### 2. 置信度计算

```python
confidence = rank_score * 0.6 + heat_score * 0.4

其中：
- rank_score = max(0, 1 - (rank - 1) / 20)  # 排名越靠前分数越高
- heat_score = min(1, heat / 10000000)       # 热度归一化
```

**置信度等级**:
- 🟢 **高置信度 (≥80%)**: 强烈推荐，大概率会火
- 🟡 **中置信度 (60-79%)**: 值得尝试
- 🔴 **低置信度 (<60%)**: 谨慎选择

### 3. 个性化匹配

系统会分析你的账号历史：
- 哪些主题阅读率高？
- 哪些内容互动多？
- 哪种标题格式效果好？

然后优先推荐符合你账号特点的话题。

## 🐛 常见问题

### Q1: 为什么推荐的主题我不感兴趣？

**原因**: 系统基于数据和算法推荐，可能不完全符合个人喜好

**解决**:
- 手动浏览推荐列表，选择合适的
- 或者自己输入主题
- 随着使用次数增加，系统会更了解你的偏好

---

### Q2: 推荐的主题太泛怎么办？

**解决**:
- 可以在推荐基础上进行微调
- 例如："AI技术" → "AI在医疗领域的3个突破"
- 更具体、更有针对性

---

### Q3: 多久更新一次推荐？

**答案**:
- 每次点击"获取推荐"都会重新获取
- 热搜数据实时更新
- 建议每天查看一次最新热点

---

### Q4: 能否禁用自动推荐？

**答案**: 可以
- 直接输入自己的主题即可
- 系统只在主题为空时才自动推荐
