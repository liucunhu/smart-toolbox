## 🔍 关键验证点

### 1. 网络请求验证
```python
# 监听发布API请求
publish_request_detected = True
response_status = 200
```

### 2. URL变化验证
```python
before_url = "https://mp.toutiao.com/profile_v4/graphic/publish"
after_url = "https://mp.toutiao.com/profile_v4/graphic/articles"
url_changed = before_url != after_url
```

### 3. 已发布作品验证（最重要）⭐
```python
# 跳转到已发布列表
await self.page.goto("https://mp.toutiao.com/profile_v4/graphic/articles")

# 搜索文章标题
article_found = await self.page.evaluate(f"""
    () => {{
        const bodyText = document.body.textContent || '';
        return bodyText.includes('{title}');
    }}
""")

if article_found:
    return {"status": "success", "message": "文章发布成功，已在已发布作品中验证"}
```

## 📊 状态判断逻辑

```
优先级1: 已发布作品中找到文章标题
         → status: "success" ✅

优先级2: 网络请求成功 (200) + URL跳转
         → status: "success" ✅

优先级3: 网络请求成功 (200) + URL未变化
         → status: "success" ✅ (异步发布机制)

优先级4: 仍在编辑页面
         → status: "draft" ⚠️

优先级5: 无成功请求 + URL未变化
         → status: "failed" ❌

其他情况:
         → status: "pending" ⏳
```

## 🔧 故障排查

### 问题1: 仍停留在发布页面
**原因**: URL未变化，但网络请求成功  
**解决**: 信任网络请求的200状态码，头条采用异步发布机制

### 问题2: 找不到确认按钮
**原因**: 对话框未完全加载  
**解决**: 增加等待时间 `await asyncio.sleep(3)`

### 问题3: 发布按钮被遮挡
**原因**: AI助手抽屉遮挡  
**解决**: 使用 `force=True` 强制点击

### 问题4: 已发布列表中找不到文章
**可能原因**:
- 文章还在审核中
- 文章被保存为草稿
- 页面加载不完整

**解决**: 检查截图，手动登录头条后台确认
