## 💡 使用指南

### 快速启动
```bash
# 1. 启动服务
docker-compose up -d

# 2. 启动后端
uvicorn main:app --reload --host 0.0.0.0 --port 8000

# 3. 启动Celery
celery -A app.tasks.celery_app worker --loglevel=info

# 4. 启动前端
cd frontend && npm run dev
```
