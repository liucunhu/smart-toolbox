## 执行步骤

1. 备份当前环境
2. 更新 requirements.txt
3. 更新 pyproject.toml
4. 重新安装依赖
5. 运行测试验证
6. 提交更改

## 注意事项

- 某些大版本升级可能需要代码适配（如 moviepy 1.x → 2.x）
- 需要先测试关键功能是否正常
- 建议分批次升级，先修复严重漏洞
