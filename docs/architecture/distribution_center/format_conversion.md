# 格式自适应转换 (Format Adaptive Conversion)

## 1. 功能描述
根据目标平台的硬性技术指标，自动将源视频转换为最佳的分辨率、比例与编码格式。

## 2. 技术实现方案

### 2.1 智能比例重绘 (Aspect Ratio Remapping)
*   **横转竖 (16:9 -> 9:16)**:
    *   **背景模糊填充**: 将原视频缩小置于中间，上下背景使用高斯模糊的原视频填充，保持画面完整性。
    *   **关键内容裁剪**: 利用 AI 识别视频中的主体（如人物），以主体为中心进行竖向裁剪，确保核心内容不丢失。
*   **竖转横 (9:16 -> 16:9)**:
    *   采用左右模糊填充或添加品牌侧边栏的方式适配宽屏。

### 2.2 平台参数标准化
*   **抖音/快手**: 
    *   分辨率：1080P (1080x1920)
    *   编码：H.264 High Profile
    *   码率：6000-8000 kbps
    *   帧率：30fps 或 60fps
*   **B站/西瓜视频**: 
    *   支持 4K 分辨率，码率提升至 15000 kbps 以上，优先使用 H.265 编码以节省带宽。
*   **小红书**: 
    *   支持 3:4 (1080x1440) 图文模式或 9:16 视频模式，强调色彩饱和度。

### 2.3 自动化处理流水线
*   **FFmpeg 滤镜链**: 编写复杂的 `filter_complex` 脚本，一次性完成缩放、裁剪、滤镜叠加与格式封装。
*   **硬件加速**: 强制调用 NVIDIA NVENC 或 Intel QSV 进行硬件编码，提升转换效率 5-10 倍。

## 3. 质量保障
*   **画质校验**: 转换完成后，通过 PSNR（峰值信噪比）与 SSIM（结构相似性）指标评估画质损耗，确保在可控范围内。
