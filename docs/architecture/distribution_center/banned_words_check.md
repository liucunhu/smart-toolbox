# 违禁词实时筛查 (Banned Words Check)

## 1. 功能描述
在内容发布前进行最后一道合规性检查，确保文案、标题及视频字幕不包含各平台明令禁止的词汇。

## 2. 技术实现方案

### 2.1 多模态内容扫描
*   **文本检测**: 
    *   利用 **AC 自动机算法** 对海量敏感词库进行高效匹配，支持毫秒级响应。
    *   覆盖范围：极限词（第一、顶级）、医疗用语、政治敏感词、引流词汇（微信、私信）。
*   **OCR 图像识别**: 
    *   对视频封面及关键帧进行 OCR 提取，识别画面中是否包含违规文字信息。

### 2.2 动态词库管理
*   **平台差异化**: 
    *   为抖音、小红书、视频号分别维护独立的词库配置文件。
    *   例如：小红书对“引流”极其敏感，而抖音对“医疗功效”审核更严。
*   **热更新机制**: 
    *   支持从云端实时更新词库，无需重启服务即可应对平台突发的规则调整。

### 2.3 智能替换建议
*   **谐音/拼音替代**: 检测到违规词后，自动提供合规的替代方案（如：“赚钱” -> “搞米” / “Z钱”）。
*   **Emoji 替换**: 将部分敏感动词替换为含义相近的 Emoji 表情。

## 3. 处理流程
1.  **输入**: 待发布的标题、文案、视频文件。
2.  **提取**: 提取视频字幕与封面文字。
3.  **匹配**: 并行执行多平台词库扫描。
4.  **反馈**: 返回违规词列表、位置索引及修改建议。
5.  **拦截**: 若存在高危违规词，直接阻断发布任务。
