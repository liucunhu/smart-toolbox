# 平台差异化文案生成 (Copywriting Generation)

## 1. 功能描述
利用大语言模型（LLM）根据目标平台的算法偏好，自动生成具有高传播潜力的视频脚本与发布文案。

## 2. 技术实现方案

### 2.1 动态提示词工程 (Dynamic Prompt Engineering)
*   **平台风格映射**:
    *   **抖音**: `Role: 悬念大师`。要求：开头 3 秒必须有冲突，每 200 字设置一个情绪高点，结尾引导评论。
    *   **小红书**: `Role: 种草达人`。要求：标题采用“二段式”，正文强制插入 Emoji，文末添加 10+ 垂直标签。
    *   **视频号**: `Role: 情感导师`。要求：侧重家庭/职场正能量，口语化表达，适合中老年群体阅读。
*   **上下文注入**: 将视频的原始主题、核心卖点以及当前的全网热搜词作为 Context 输入给 LLM。

### 2.2 爆款元素植入逻辑
*   **黄金 3 秒原则**: AI 优先生成具有视觉冲击力或认知反差的开场白。
*   **关键词堆砌优化**: 针对 SEO 需求，在文案中自然融入长尾搜索词（如“2026最新玩法”）。
*   **互动引导语**: 根据不同平台的调性，自动附加“点赞收藏防走失”或“评论区聊聊你的看法”等引导语。

### 2.3 违禁词预过滤
*   在 LLM 输出后，立即通过本地敏感词库进行第一轮扫描，将极限词（如“第一”、“顶级”）替换为合规的表达方式。

## 3. 处理流程
1.  **输入**: 原始素材主题 + 目标平台。
2.  **检索**: 获取该平台最新的爆款文案模板与热词。
3.  **生成**: LLM 产出初稿。
4.  **优化**: 风格微调与违禁词替换。
5.  **输出**: 结构化文案（标题 + 正文 + 标签）。
