# 智能去重与伪原创 (Intelligent Deduplication)

## 1. 功能描述
针对多平台分发场景，对原始视频进行深度加工，规避平台的查重机制（MD5 校验、指纹比对、画面抽帧），确保作品被判定为“原创”。

## 2. 技术实现方案

### 2.1 画面层去重 (Visual Layer)
*   **几何变换**: 
    *   **随机镜像**: 以 50% 的概率对视频进行水平翻转。
    *   **微缩放**: 将画面放大 1.05-1.1 倍，并随机改变裁剪位置。
*   **色彩扰动**: 
    *   叠加透明度极低（5%-10%）的噪点滤镜或色彩平衡调整。
    *   **画中画 (PIP)**: 在背景添加模糊处理的同款视频，中间放置主视频，改变画面布局。

### 2.2 音频层去重 (Audio Layer)
*   **变速变调**: 
    *   将音频速度微调 ±0.1 倍，同时使用声码器调整音调，保持听感自然但波形改变。
*   **背景音混合**: 
    *   混入低分贝的环境音（如风声、街道声）或平台热门 BGM，覆盖原始音频指纹。

### 2.3 数据层与结构层 (Data & Structural Layer)
*   **元数据清洗**: 
    *   彻底擦除视频文件的 EXIF、编码器版本、创建时间等元数据。
    *   重新封装容器格式（如从 MP4 转为 MOV 再转回 MP4）。
*   **AI 结构重组**: 
    *   利用 NLP 技术分析文案逻辑，对于非强时序依赖的片段（如展示多个产品），自动打乱播放顺序。
    *   **插帧/抽帧**: 每隔 50 帧随机插入一帧重复帧或删除一帧，破坏原有的帧序列哈希值。

## 3. 去重效果验证
*   **自检验证**: 系统在发布前会将处理后的视频与库内已有视频进行相似度比对，确保相似度低于平台阈值（通常为 60%-70%）。
