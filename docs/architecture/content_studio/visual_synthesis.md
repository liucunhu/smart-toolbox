# 视觉爆款合成 (Visual Synthesis)

## 1. 功能描述
通过计算机视觉技术，自动为视频生成高点击率的封面、动态字幕及特效，提升作品的视觉吸引力。

## 2. 技术实现方案

### 2.1 智能封面工厂 (Smart Cover Factory)
*   **关键帧提取**: 
    *   使用 OpenCV 分析视频帧的亮度、对比度及人脸清晰度，筛选出最具代表性的 3-5 个候选帧。
    *   **场景识别**: 识别视频内容（如：美妆、美食、风景），自动匹配对应的滤镜风格。
*   **模板化合成**:
    *   **小红书模式**: 采用 3:4 比例，叠加 ins 风边框与“大字报”标题。
    *   **抖音模式**: 采用 9:16 比例，支持三格拼接或高饱和度人物特写。

### 2.2 情绪字幕系统 (Emotional Subtitles)
*   **ASR 语音识别**: 利用 Whisper 模型将视频语音高精度转为文字，并打上时间戳。
*   **情感分析驱动**: 
    *   对每一句文案进行情感打分。
    *   **高潮部分**: 自动放大字号、改变字体颜色（如黄色/红色）并添加抖动特效。
    *   **平淡部分**: 保持常规样式，确保阅读舒适度。

### 2.3 热门 BGM 匹配
*   **节奏对齐**: 分析视频的剪辑点（Cut points），从音乐库中选取节奏点（Beats）与之匹配的背景音乐。
*   **音量 ducking**: 当检测到语音时，自动降低 BGM 音量，确保人声清晰。

## 3. 性能优化
*   **GPU 加速**: 所有的视频渲染与滤镜处理均通过 FFmpeg 调用 GPU 硬件加速。
*   **异步处理**: 封面生成与字幕合成并行执行，减少整体等待时间。
