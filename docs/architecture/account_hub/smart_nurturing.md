# 智能养号系统 (Smart Nurturing System)

## 1. 功能描述
通过模拟真人用户的浏览与互动行为，提升新注册账号的初始权重，降低被平台判定为营销号的风险。

## 2. 技术实现方案

### 2.1 拟人化行为调度器
*   **停留时长算法**: 
    *   采用正态分布 $N(\mu, \sigma^2)$ 生成视频观看时长，模拟用户从“快速划过”到“深度观看”的自然分布。
    *   引入随机休眠机制，在两次操作之间插入 5-30 秒的无操作时间。
*   **轨迹模拟**: 
    *   使用贝塞尔曲线模拟鼠标滑动与手机触屏滑动轨迹，避免直线移动。

### 2.2 领域垂直化训练
*   **关键词匹配**: 
    *   根据账号预设的定位（如：美妆、科技），爬取平台热门话题标签。
    *   优先浏览、点赞带有相关标签的内容，强化账号的垂直领域标签。
*   **互动概率模型**: 
    *   **点赞**: 5% 概率（针对高相关性内容提升至 15%）。
    *   **评论**: 1% 概率（调用 LLM 生成简短、正向的通用评论）。
    *   **转发**: 0.5% 概率。

### 2.3 活跃时段控制
*   **时间窗口**: 仅在平台流量高峰期（如 12:00-13:00, 20:00-22:00）执行高强度互动。
*   **休息模拟**: 每日设置 6-8 小时的“睡眠时间”，期间不产生任何日志。

## 3. 配置示例
```yaml
nurturing_config:
  duration_days: 7
  daily_actions:
    browse_count: 50
    like_probability: 0.05
    comment_probability: 0.01
  active_hours: ["12:00-13:00", "20:00-22:00"]
```
