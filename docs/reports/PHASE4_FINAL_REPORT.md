## 🚀 使用演示

### 场景 1: 分解并执行任务

1. **输入目标**
   ```
   创作一篇关于AI的文章并发布到头条
   ```

2. **点击"分解目标"**
   - 系统自动分解为 6 个子任务

3. **执行任务**
   - 选项 A: 点击单个任务的"执行"按钮
   - 选项 B: 点击"🚀 执行所有任务"批量执行

4. **查看结果**
   - 每个任务卡片显示执行状态
   - 完成后显示"查看结果"按钮
   - 结果以 JSON 格式展示在卡片中

### 场景 2: 监控执行情况

1. **查看执行历史**
   - 滚动到页面底部
   - 查看最近 20 条执行记录
   - 点击"刷新"获取最新数据

2. **查看统计数据**
   - 右侧统计面板显示各智能体的成功率
   - 进度条直观展示性能
   - 详细数字显示任务数量
