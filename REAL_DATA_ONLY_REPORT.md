## ⚠️ 注意事项

### 1. 失败处理

当无法获取真实数据时：
- ✅ **返回空列表** `[]`
- ✅ **记录错误日志** `logger.error()`
- ❌ **不再使用模拟数据**

### 2. 可能的失败原因

- 网络问题
- API限流
- 反爬机制
- 页面结构变化
- 服务暂时不可用

### 3. 前端展示

前端需要处理空数据的情况：
```javascript
if (response.data.total === 0) {
  // 显示提示："暂无数据，请稍后重试"
  // 而不是显示模拟数据
}
```
